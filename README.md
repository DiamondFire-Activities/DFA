# DFA

DiamondFire Activities repository. This repository is *private* access, be mindful about sharing any information. Maintained by **Refrizor**


## Development
The project solely uses Java, with some use cases of JSON for world generation and YAML for configuration. It depends on the latest stable version of Spigot (run BuildTools.jar), and requires mysql-connector-java's latest stable version. DFA runs on JDK 17.
